API Reference


        

          
Base URL

          

            https://cultscreener-api.onrender.com/v1
          

        


        

          
Authentication

          
Include your API key in every request as a header:

          

            X-API-Key: your_api_key_here
          

        


        

          
GET /leaderboard

          
Returns the ranked conviction leaderboard with optional filters.

          

            
              

                Parameter
                Type
                Description
              

            
            
              

                limit
                number
                Results per page (default 25, max 100)
              

              

                offset
                number
                Pagination offset (default 0)
              

              

                minConviction
                number
                Filter by minimum conviction score (0–100)
              

              

                minMcap
                number
                Filter by minimum market cap (USD)
              

              

                maxMcap
                number
                Filter by maximum market cap (USD)
              

              

                minSample
                number
                Filter by minimum wallet sample size
              

            
          

          

            curl -H "X-API-Key: YOUR_KEY" \
  "https://cultscreener-api.onrender.com/v1/leaderboard?limit=10&minConviction=50"
          

        


        

          
GET /leaderboard/:mint

          
Returns conviction data and leaderboard rank for a specific token.

          

            
              

                Parameter
                Type
                Description
              

            
            
              

                mint
                string
                The token's Solana mint address (path parameter)
              

            
          

          

            curl -H "X-API-Key: YOUR_KEY" \
  "https://cultscreener-api.onrender.com/v1/leaderboard/So11111111111111111111111111111111111111112"
