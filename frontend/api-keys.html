📖 API Documentation


        
Base URL

        

          https://cultscreener-api.onrender.com/v1
        


        
Authentication

        
Include your API key in the request header:

        

          X-API-Key: your_api_key_here
        


        
Endpoints


        

          

            
GET

            
/leaderboard

          

          
Get the conviction token leaderboard

          
Query Parameters:

          

            
limit - Results per page (max 100, default 25)

            
offset - Pagination offset (default 0)

            
minConviction - Filter by minimum conviction % (0-100)

            
minMcap - Filter by minimum market cap

            
maxMcap - Filter by maximum market cap

            
minSample - Filter by minimum sample size

          

          
Example:

          

            curl -H "X-API-Key: your_key" "https://cultscreener-api.onrender.com/v1/leaderboard?limit=10&minConviction=50"
          

        


        

          

            
GET

            
/leaderboard/:mint

          

          
Get conviction data for a specific token with its leaderboard rank

          
Path Parameters:

          

            
mint - The token's mint address (Solana address)

          

          
Example:

          

            curl -H "X-API-Key: your_key" "https://cultscreener-api.onrender.com/v1/leaderboard/EPjFWaLb3odcccccccccccccccccccccccccccccccccc"
          

        


        
Response Format

        
All responses return JSON with the following structure:

        

          {
  "success": true,
  "data": [...],
  "limit": 25,
  "offset": 0,
  "total": 1234
}
        


        
Error Handling

        
Invalid API key or rate limiting will return appropriate HTTP status codes:

        

          
401 - Invalid API key

          
403 - API key revoked or inactive

          
429 - Rate limit exceeded

          
404 - Token not found
